# Retell CLI

A TypeScript CLI tool for managing Retell AI agents across workspaces with composable prompts and file-based configuration.

## Installation

```bash
npm install
```

## Development

```bash
# Install dependencies
npm install

# Run type checking
npm run type-check

# Run linter
npm run lint

# Run tests
npm test

# Run tests in watch mode
npm run test:watch

# Build
npm run build
```

## Testing Strategy

- **Unit Tests**: `npm run test:unit` - Test individual modules in isolation
- **Integration Tests**: `npm run test:integration` - Test module interactions
- **E2E Tests**: `npm run test:e2e` - Test complete workflows
